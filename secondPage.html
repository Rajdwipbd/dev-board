Question-1: What are the different ways to select an element in the DOM?

We can select an element in the DOM by using getElementById(),getElementsByClassName(),getElementsByTagName(),querySelectorAll() and querySelector() .

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML sets or gets HTML content, allowing tags. innerText retrieves only visible text, ignoring hidden elements. textContent gets all text, including hidden ones, without formatting. innerHTML poses security risks due to XSS, while textContent is safer. Use innerHTML for HTML, textContent for plain text.

Question-3: What is event delegation in the DOM?

Event delegation is a method where a parent element controls events for its child elements. Instead of adding event listeners to each child, you add one listener to the parent. This makes the code faster and helps manage events for new elements added later.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event starts at the target element and then moves up to its parent elements

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element in JavaScript, use document.createElement('tagName'). To add it to the DOM, use parentElement.appendChild(newElement). To remove an element, use parentElement.removeChild(childElement)
